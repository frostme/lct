# lct remove

Remove modules from an LCTFile or globally

## Usage

```bash
lct remove [MODULE] [OPTIONS]
```

## Examples

```bash
lct remove DannyBen/approvals.bash
```

```bash
lct remove -g DannyBen/approvals.bash
```

## Arguments

#### *MODULE*

GitHub owner/repo of the module to remove

## Options

#### *--global, -g*

Use global module removal from config.yaml
