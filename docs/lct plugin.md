# lct plugin

Develop and manage lct plugins

## Usage

```bash
lct plugin COMMAND
```

## Examples

```bash
lct plugin create myplugin
```

## Commands

- [create](lct%20plugin%20create.md) - Create a new lct plugin
- [publish](lct%20plugin%20publish.md) - Publish an lct plugin to the registry
- [install](lct%20plugin%20install.md) - Install an lct plugin from the registry
- [remove](lct%20plugin%20remove.md) - Remove an installed lct plugin
- [list](lct%20plugin%20list.md) - List all installed lct plugins
