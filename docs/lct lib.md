# lct lib

Manage lct libraries

## Usage

```bash
lct lib COMMAND
```

## Examples

```bash
lct lib add mylib
```

## Commands

- [install](lct%20lib%20install.md) - Install libraries
- [add](lct%20lib%20add.md) - Add a library to lct
- [remove](lct%20lib%20remove.md) - Remove a library from lct
- [dump](lct%20lib%20dump.md) - Dump the current lct libraries to your Brewfile
- [list](lct%20lib%20list.md) - List all lct libraries
