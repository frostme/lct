# lct alias add

add ALIAS alias

## Usage

```bash
lct alias add ALIAS COMMAND
```

## Arguments

#### *ALIAS*

Name of the alias to add

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *COMMAND*

Command to execute for the alias

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes
