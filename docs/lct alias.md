# lct alias

Manage lct aliases

## Usage

```bash
lct alias [COMMAND]
```

## Examples

```bash
lct alias
```

```bash
lct alias add ll 'ls -la'
```

```bash
lct alias remove ll
```

## Commands

- [add](lct%20alias%20add.md) - add ALIAS alias
- [remove](lct%20alias%20remove.md) - unset and remove ALIAS alias
