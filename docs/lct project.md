# lct project

Manage tracked code projects

## Usage

```bash
lct project COMMAND
```

## Examples

```bash
lct project list
```

```bash
lct project add
```

```bash
lct project remove
```

## Commands

- [list](lct%20project%20list.md) - List configured projects
- [add](lct%20project%20add.md) - Add the current git project to the config
- [remove](lct%20project%20remove.md) - Remove the current git project from the config
