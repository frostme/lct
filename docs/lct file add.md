# lct file add

Add FILE mapping to other list

## Usage

```bash
lct file add FILE SOURCE
```

## Arguments

#### *FILE*

Destination path to store in remote repository (relative to other/)

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *SOURCE*

Source file path on local machine (relative to HOME or ~/...)

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes
