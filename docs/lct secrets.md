# lct secrets

Manage encrypted secret paths

## Usage

```bash
lct secrets COMMAND
```

## Examples

```bash
lct secrets add ~/.claude.json
```

```bash
lct secrets remove ~/.claude.json
```

```bash
lct secrets list
```

## Commands

- [add](lct%20secrets%20add.md) - Add PATH to the secrets list (stored encrypted in gather)
- [remove](lct%20secrets%20remove.md) - Remove PATH from the secrets list
- [list](lct%20secrets%20list.md) - List configured secret paths
