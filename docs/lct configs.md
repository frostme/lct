# lct configs

Manage configs list

## Usage

```bash
lct configs COMMAND
```

## Examples

```bash
lct configs list
```

```bash
lct configs add alacritty
```

```bash
lct configs remove alacritty
```

## Commands

- [add](lct%20configs%20add.md) - Add CONFIG to configs list
- [remove](lct%20configs%20remove.md) - Remove CONFIG from configs list
- [list](lct%20configs%20list.md) - List configured configs
