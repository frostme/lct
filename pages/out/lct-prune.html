lct prune

Prune config and plugins files

Usage

bash lct prune [OPTIONS]

Examples

bash lct prune

bash lct prune --plugins

bash lct prune --plugins --unused --dry

bash lct prune -pud

Options

–plugins, -p

Prune unused plugins

–unused, -u

Prune unused config files

–cache, -c

Prune cache files

–dry, -d

Show all files that would be pruned
